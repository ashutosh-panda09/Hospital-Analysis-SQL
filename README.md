# Hospital Data Analysis Using PostgreSQL

## Project Overview

This project is about looking at hospital data using PostgreSQL.
The goal of the Hospital Data Analysis project is to understand how hospitals work by using queries and finding useful information about Hospital Data Analysis, including patients, departments, doctors, admissions, discharges and medical expenses.
The Hospital Data Analysis project shows how to use SQL for cleaning data making tables asking questions and looking at business information.

---

## Objectives

- Make a database for hospital information. Organize it
- Keep track of hospital records and manage them
- Use SQL queries to analyze Hospital Data Analysis
- Answer important questions about the hospital data
- Show the results in a clear and easy to understand way

---

## Tools Used

- PostgreSQL
- SQL
- pgAdmin

---

## Dataset Features

The hospital data includes the following:
- Hospital Name
- Hospital Location
- Department
- Number of doctors
- Number of patients
- Date patients are admitted
- Date patients are discharged
- Medical expenses for Hospital Data Analysis

---

## Key Analysis Performed

This Hospital Data Analysis project includes queries about:
- number of patients in the hospital
- Looking at patients by department
- Comparing doctors and patients
- Trends in admissions and discharges
- Highest and lowest medical expenses for Hospital Data Analysis
- How well each hospital is doing
- Filtering and calculating based on date
- Using aggregate and conditional SQL operations, for Hospital Data Analysis
